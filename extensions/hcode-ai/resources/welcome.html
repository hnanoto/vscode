HCode AI

O assistente de IA mais completo de um IDE open source

💬 Como usar os agentes
💬

Painel de Chat

Clique no ícone de chat na barra lateral ou pressione Ctrl+Alt+I para abrir o chat e digitar suas perguntas.

⌨️

Comandos de Chat

No chat, use slash commands como /agent, /fix, /commit para ativar agentes específicos.

🖱️

Menu de Contexto

Selecione código, clique com botão direito → HCode AI → Explicar, Corrigir, Gerar Testes ou Refatorar.

📊

Status Bar

Clique em 🤖 Ollama · qwen2.5 na barra de status para trocar de provider ou modelo instantaneamente.

⚡

Paleta de Comandos

Pressione Ctrl+Shift+P e pesquise "HCode AI" para ver todos os comandos disponíveis.

🧬

SCM / Git

Na aba de controle de versão, clique em ⚡ ao lado da mensagem de commit para gerar a mensagem automaticamente.

🤖 Todos os Agentes Disponíveis
Comando O que faz Tipo
@hcode [pergunta] Agente principal — responde perguntas, explica código, lê arquivos, busca no projeto Padrão
/agent [tarefa] Modo Agente Completo — executa tarefas de engenharia autônomas: lê arquivos, edita código, roda testes, verifica erros Autônomo
/fix Analisa o código selecionado ou o arquivo atual e corrige bugs, erros de tipo e problemas de lógica Padrão
/commit Gera uma mensagem de commit semântica, no padrão Conventional Commits, para o diff atual Git
/test Gera testes unitários para o código selecionado ou função ativa Padrão
/explain Explica em detalhes o código selecionado, incluindo lógica, padrões e possíveis melhorias Padrão
/setup Abre o wizard de configuração para escolher o provider e modelo de IA Config
🔧 Ferramentas que os Agentes Usam Automaticamente
FerramentaO que faz
hcode_readFileLê qualquer arquivo do workspace (com segurança anti-traversal)
hcode_listDirectoryLista estrutura de pastas e arquivos do projeto
hcode_runTerminalExecuta comandos shell — npm, git, testes, build
hcode_searchCodeBusca padrões no código com ripgrep embutido
hcode_gitStatusRetorna branch, status e histórico de commits
hcode_getDiagnosticsLista erros e warnings do TypeScript/LSP
hcode_webSearchBusca na internet via DuckDuckGo — sem API key necessária
hcode_inlineEditEdita código diretamente no arquivo — sem copiar e colar
hcode_createFileCria novos arquivos com conteúdo gerado pelo AI
